MAP = [[],[],[],[],[]] -> variable qui représente l'état des différents elm de la partie
ETAT_PARTIE = [typedetour, joueur]

---------------- PARTIE LOGIQUE ------------------------------

Les fonctions pour placer le jeton:
    - une fct qui détermine la couleur/nature du jeton en fonction de la partie (j1, j2)    __ entrée : ETAT_PARTIE ; sortie : couleur
    - une fct qui détermine si la case est bien libre   __ entrée : position(x, y) ; sortie : True / False
    - une fct qui actualise la carte de jeu    __ entrée : position(de l'action) ; sortie: MAP actualisé

Les fonctions pour déplacer un jeton:
    - une fct qui détermine si le joueur qui déplace détiens le jeton   __ entrée : ETAT_PARTIE ; sortie : couleur
    - une fct qui détermine si une place est libre      __ entrée : position(x, y) ; sortie : True / Fals
    - une fct qui fait le déplacement en fonction de la ligne cliqué (si les deux conditions sont validés)      __ entrée : position(de l'action) ; sortie: MAP actualisé

Fonction qui renvois l'élément sur lequel le joueur a cliqué:
    - fonction qui renvois une liste [x, y] par coordonnée d'exclusion/inclusion     __ entrée : event(x, y) ; sortie : MAP[x][y]

Fonction main (de jeu):
    - gère les tours, appel la fonction de gui et fais une boucle       __ None
